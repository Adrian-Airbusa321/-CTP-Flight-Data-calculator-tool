# -CTP-Flight-Data-calculator-tool

Python script that calculates flight time and fuel burn for CTP pairs. 
The feature to put in the distance yourself is going to be implemented in the near future.

Run the .py script in .../Tool (with Python!)
Enter the parameters that the python script prompts you to enter. For aircraft types and airports all options and formats are listed, for weight and wind data just enter the numerical value.

Route data in courtesy of @ianolaf on discord.

Intallation: 
Install the zip file and unpack it while preserving the folder structure as it follows:
