# -CTP-Flight-Data-calculator-tool

Python script that calculates flight time and fuel burn for all transatlantic eoutes with airports <1 Million monthly pax
The feature to put in the distance yourself is also implemented

Run the .py script in .../Tool (with Python!)
Enter the parameters that the python script prompts you to enter. For aircraft types and airports all options and formats are listed, for weight, reserve fuel and wind data just enter the numerical value.

CTP Route data in courtesy of @ianolaf on discord.
Other transatlantic Data collected and calculated by Opus 4.7. Triple checked independetly, but AI can always make mistakes.
